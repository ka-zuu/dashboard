# Smart Home Dashboard Agent Context

> [!IMPORTANT]
> **CRITICAL INSTRUCTIONS**
> - **ALL Output Must Be in Japanese**: Every interaction, comment, documentation, Walkthrough, Implementation Plan, and commit message MUST be written in Japanese.
> - **Walkthroughs**: These are critical for the user to understand the changes. They MUST be written in Japanese.
> - **Exceptions**: Variable names, function names, and code syntax should remain in English.

## コーディング規約
- **言語:** コメント、ドキュメント、コミットメッセージ、Walkthrough、Implementation Planなど、全ての出力は**必ず日本語**で行う。
- **チャット:** ユーザーとのやり取りは全て日本語。
- **命名:** 変数・関数名は英語。
- **モジュール:** ES Modules (`import`/`export`) を使用。
- **スタイル:** Prettier/ESLintの標準設定に従う。

## 仕様書
詳細な仕様は [SPEC.md](SPEC.md) を参照すること。変更がある場合は必ずSPEC.mdを更新すること。
