# Smart Home Dashboard Agent Context

## コーディング規約
- **言語:** コメント、ドキュメントは全て日本語。
- **チャット:** ユーザーとのやり取り、コメント、Implementation Plan, Walk Through等は必ず日本語で行う。
- **コミットメッセージ:** 必ず日本語で記述する。
- **命名:** 変数・関数名は英語。
- **モジュール:** ES Modules (`import`/`export`) を使用。
- **スタイル:** Prettier/ESLintの標準設定に従う。
